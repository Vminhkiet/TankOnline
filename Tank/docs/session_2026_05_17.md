## 3. Số liệu hiệu năng — Giải thích

- **tick avg** = trung bình wall-clock time mỗi tick (60 Hz). Ổn định, ít bị ảnh hưởng spike.
- **tick p99** = percentile 99 — 1% tick chậm nhất. Phản ánh OS scheduling jitter, spike khi tạo match mới.
- **Overruns** = tick vượt 16,667µs. Ảnh hưởng trực tiếp gameplay (frame drop phía client).
- **418µs/match/tick** = derived value: `tick_avg × N_cores ÷ N_matches` = 1,670µs × 8 ÷ 32. Server KHÔNG đo trực tiếp per-match time.

## 6. isolate_cores.ps1 — Script cô lập môi trường

File: `test/isolate_cores.ps1`. Chạy với quyền Admin trên Windows trước khi benchmark.

6 bước:
1. High Performance power plan (tắt CPU throttling)
2. Disable Windows Defender real-time (tránh scan spikes)
3. Pin tất cả background processes vào cores 0–3 (mask 0x0F)
4. Pin server_tank vào cores 4–7 (mask 0xF0), priority = RealTime
5. Set PowerShell session = High priority
6. Stop Windows Update service

Restore: `powershell -File isolate_cores.ps1 -Restore`
